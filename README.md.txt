# Translate Language Project - assignment
## Project build using laravel API

## Features
- Translate your text to any language
- Google Translation API Integrated


## Steps for uses
## 1] Prerequisite
Google Translation API key , [click to know](https://weglot.com/blog/google-translate-api-key/)
## 2] Copying env file
```sh
copy env.example To .env
```

## 3] Installation
Install the dependencies and devDependencies 
Run below command in project root folder
```sh
composer install 
```

## 3] Running server
Run below command in project root folder
```sh
php artisan serve 
```
